# term_project
this is my term project for our school's programming class, I made a short AES encryption. 
AES 是一種對稱式加密，它的優點是非線性、快速且具高擴散性，廣泛使用於各種端對端加密，使用方式很簡單，改動程式中的text和key就可以跑動了，因為這和網路上的AES幾乎一樣，所以會得到一樣的加密結果，並且也可確認解密後和原本訊息一致。程式包含encryption和decryption兩大功能。在開發過程中大部分時間我都在搞懂一些數學式、代換和排列的方式，程式的原理並不複雜，但要用程式重現想法難度就差了不只一點。
